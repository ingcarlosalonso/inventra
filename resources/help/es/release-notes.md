# Novedades del sistema

Cada vez que se publica una nueva versión de In-ventra, aparece automáticamente un popup con las mejoras incluidas en ese release.

## ¿Cuándo aparece el popup?

El popup se muestra la primera vez que ingresás al sistema después de que se publicó una nueva versión. Una vez que lo cerrás, no vuelve a aparecer.

## ¿Qué información muestra?

El popup agrupa las novedades por tipo:

- **Novedad** — nuevas funcionalidades agregadas al sistema
- **Mejora** — mejoras sobre funcionalidades existentes
- **Corrección** — errores corregidos
- **Seguridad** — actualizaciones relacionadas con la seguridad

## ¿Puedo volver a verlo?

Por el momento, el popup solo se muestra una vez por versión. Si querés revisar las novedades, podés consultarlas en esta sección de ayuda.
