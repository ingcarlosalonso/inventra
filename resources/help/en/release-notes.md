# What's new

Every time a new version of In-ventra is published, a popup appears automatically with the improvements included in that release.

## When does the popup appear?

The popup is shown the first time you log in after a new version has been published. Once you dismiss it, it won't appear again.

## What does it show?

The popup groups changes by type:

- **New feature** — new functionality added to the system
- **Improvement** — enhancements to existing functionality
- **Bug fix** — errors that have been corrected
- **Security** — security-related updates

## Can I see it again?

Currently the popup only shows once per version. If you want to review what changed, you can check this help section.
